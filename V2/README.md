# V2 - Analytics, SEO & Instrumentation Bundle

This bundle contains a ready-to-drop analytics wrapper, provider examples (Plausible & PostHog),
SEO templates (robots, sitemap config, OG meta sample), tracking events docs, and a theme instrumentation
example that your theme bundles can call.

Files:
- analytics/lib/analytics.js
- analytics/providers/plausible.js
- analytics/providers/posthog.js
- theme-instrumentation/initTheme.js
- seo/robots.txt
- seo/next-sitemap.config.js
- seo/og-template.html
- seo/privacy-footer.md
- docs/tracking-events.md
